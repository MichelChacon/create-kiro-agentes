# Diagnosticar un problema

Usa este prompt cuando algo no funciona.

---

Diagnostica este problema:
[describe el error]

Contexto:
- Archivo: [ruta]
- Cuando ocurre: [trigger]
- Lo que espero: [comportamiento correcto]
- Lo que obtengo: [error exacto]

El agente analiza, identifica causa raiz, propone solucion y verifica.
